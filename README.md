# Portfolio Web Personal - Josep Perez

Este proyecto es mi portafolio personal desarrollado como parte del módulo **M0373 - Llenguatges de Marques** del ciclo CFGS Administración de Sistemas Informáticos en Red (ASIX).

## 🚀 Sobre el proyecto
En esta web presento mis habilidades como desarrollador web, mostrando proyectos interactivos y un diseño moderno adaptado a dispositivos móviles.

### ✨ Funcionalidades destacadas
El proyecto incluye varias funcionalidades técnicas avanzadas:
1.  **Validación de Formulario:** Sistema en JavaScript que valida el correo y muestra errores personalizados en catalán.
2.  **Filtros de Proyectos:** Botones interactivos para clasificar los proyectos por categoría (Web, App, Branding).
3.  **Botón "Scroll to Top":** Elemento de navegación que aparece dinámicamente al hacer scroll.
4.  **Proyectos Interactivos:** Implementación funcional de un **Tetris** y una **Calculadora** usando lógica de programación.

## 🛠️ Tecnologías utilizadas
* **HTML5:** Estructura semántica.
* **CSS3:** Diseño responsivo, Flexbox, Grid y variables CSS.
* **JavaScript (ES6):** Lógica del DOM y control de eventos.
* **VS Code:** Entorno de desarrollo.
